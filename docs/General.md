# EL 比赛项目设计文档

## 总体框架

本游戏采用两层的游戏逻辑，宏观层面模拟病毒在一个国家中的传播过程及其造成的社会影响，微观层面以小游戏的形式表现病毒入侵人体的过程。同时两个层面相互影响，以达到增强交互性的效果。
